# MyDigitalLibrary

A personal academic library app that helps students store, organise and find their study materials in one place.

## The Problem

As a student, study materials pile up fast. Class notes, past questions, PDF readings, all scattered across different folders, WhatsApp chats and email attachments. Finding the right material when you need it is frustrating and time-wasting.

## The Solution

MyDigitalLibrary is a personal academic library that lets students store and organise their study materials by level, course and topic. Materials can be searched by keyword or filtered by level and course for quick access.

## Features

- **Dashboard** — view all your study materials in one organised place
- **Search** — find materials quickly by typing a keyword
- **Filter** — narrow down materials by academic level or course
- **Upload** — add new materials with title, course, topic, level and type
- **Persistent storage** — materials saved in browser localStorage across sessions



## Tech Stack

- **React** — frontend UI and component structure
- **React Router** — page navigation
- **JavaScript** — filtering, search logic and localStorage management
- **CSS** — custom styling.
- **localStorage** — browser-based data persistence (no backend required)

## Getting Started

To run this project locally:

```bash
git clone https://github.com/Wura-Ola-Eng/MyDigitalLibrary.git
cd MyDigitalLibrary/client
npm install
npm start
```

The app will open at `http://localhost:3000`
