# MyDigitalLibrary

A personal academic library app that helps students store, organise and find their study materials in one place.

## The Problem

As a student, study materials pile up fast. Class notes, past questions, PDF readings, all scattered across different folders, WhatsApp chats and email attachments. Finding the right material when you need it is frustrating and time-wasting.

## The Solution

MyDigitalLibrary gives you one organised place to store and find all your study materials. You can filter by level, filter by course, or search by keyword to find exactly what you need in seconds.

## Features

- Browse all your study materials on a clean dashboard
- Search materials by title
- Filter by academic level and course
- Upload new materials with course, topic, level and type
- Materials remain in your browser even after you close the page
- Fully responsive on desktop and mobile

## Tech Stack

- **React** — frontend UI and component structure
- **React Router** — page navigation
- **JavaScript** — filtering, search logic and localStorage management
- **CSS** — custom styling.
- **localStorage** — browser-based data persistence (no backend required)

## Getting Started

To run this project locally:

```bash
git clone https://github.com/Wura-Ola-Eng/MyDigitalLibrary.git
cd MyDigitalLibrary/client
npm install
npm start
```

The app will open at `http://localhost:3000`

## What's Next
